Choose your network

Testnet is free and risk-free. Mainnet uses real BNB. You can switch later by re-running this wizard.

Default testnet: bsc-testnet public RPCs. Mainnet: bsc-dataseed.
x402 works without a key (USDC micro-payments per request). Pro API key gives you a free tier.

Create or import a wallet

The wallet is encrypted with AES-256-GCM at ~/.twak/wallet.json using a password you choose. The key never leaves this host.

Paste your 32-byte hex private key. Encrypted to disk immediately; never logged or sent over the wire.

Sign the User Policy

The policy is the only thing you sign. Every trade the agent makes is checked against this signed policy before it goes on-chain.

◆
The wallet you just created will sign config/policy.yaml with EIP-191. You can re-sign any time to bump the policy version.
In single-user setups, leave blank. In multi-sig setups, set this to your cold-wallet evaluator.

7-Day Replay python -m backtest.replay

The replay harness drives the same three sleeves (A/B/C) through a recorded 7-day tape and emits a full PnL report. Run it from a terminal: 

bash bnbagent --replay

The HTML report lands at data/reports/replay.html and includes Sharpe, Sortino, Calmar, max drawdown, hit rate, and sleeve attribution.
